# Project Overview
This project contains configurations, guidelines, and instructions for various AI code agents used in our development workflow.

The project is basically scaffolding project sctructure stored in `/dist` folder that can be imported into other projects to quickly set up AI code agents.

All the prompts and instructions in this project are relating to the typical python / machine learning development projects.




## The Agents Frameworks
- Github Copilot
- OpenCode AI
- OpenAI Codex
- Warp.Dev AI
